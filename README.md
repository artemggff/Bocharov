# Описание проетка

*"Это программа на Python с графическим интерфейсом (Tkinter), которая работает как генератор случайных цитат.

Основные функции:
Генерация: показывает случайную цитату из встроенной базы.
История: сохраняет все просмотренные цитаты в отдельный список.
Фильтрация: позволяет искать цитаты в истории по автору или теме.
Управление данными: можно добавлять новые цитаты в базу, а также сохранять и загружать историю из JSON-файла."#

## Требования

* Python 3.6+
* Библиотеки: `tkinter` (встроен в Python), `json` (встроен в Python)
