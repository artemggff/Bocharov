# Описание проетка

*"Training Planner — это GUI‑приложение для планирования тренировок с возможностью:
* добавления тренировок с указанием даты, типа и длительности;
* фильтрации по типу тренировки и дате;
* сохранения данных в JSON‑файл;
* загрузки данных при запуске приложения."#
